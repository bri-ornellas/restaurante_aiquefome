Atividade:

O dono do restaurante Äi que Fome” contratou você para criar um site para seu restaurante, no entanto, ele não tem ideia de como fazer o site. Desta maneira você deverá criar um protótipo como o exemplo que será exibido para mostrar a ele.

Procedimentos:

Crie um repositório com o nome “restaurante_aiquefome”.

Crie os diretórios para organização dos arquivos. Exemplos: paginas, imagens, etc.

Adicione uma subcategoria "Sobremesas" no menu contendo 3 opções aleatórias de sobremesas, este elemento será da lista principal ordenada (<ol>).

Para cada elemento da lista secundária deverá ter um link para uma imagem de 150px por 150px do produto/comida escolhida. 

Conteúdo Referencial:

Os Elementos html utilizados para fazer a página são:( em caso de dúvida pesquise sobre os elementos);
Cabeçalho de nível 1;
Cabeçalho de nível 2;
Lista Ordenada;
Lista Não Ordenada;
Lista Alinhada;
Tabela com linhas e colunas expandidas/mescladas;
Imagens;
Links.

Utilize as tags semânticas de organização de conteúdo do html ex: header, main, footer.
